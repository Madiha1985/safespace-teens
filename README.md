# safespace-teens
SafeSpace Teens is a social-learning platform designed for teenagers aged 12–17.
The platform provides safe chatrooms, study groups, reading clubs, and a private
feeling journal to support learning, self-expression, and positive peer interaction.

## Features (MVP)
- Topic-based chatrooms (study & reading rooms)
- Real-time messaging using Socket.io
- Reading hub for book reviews and discussions
- Private feeling journal
- User authentication
- Moderation tools (basic filtering & reporting)

## Tech Stack
- Frontend: Next.js (React)
- Backend: Express.js
- Database: MongoDB + Mongoose
- Real-time: Socket.io
- Testing: Jest
- Deployment: Vercel (frontend), Render/Railway (backend)
